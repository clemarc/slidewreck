# TalkForge Documentation

TalkForge is a multi-agent AI pipeline that transforms a topic and constraints into a complete conference talk package — research brief, narrative outline, speaker script, slide specifications, and speaker prep kit. Built on [Mastra](https://mastra.ai/) with human-in-the-loop approval gates at every stage.

## Methodology

This project follows the [BMAD methodology](https://github.com/bmadcode/BMAD-METHOD) — spec-driven development where stories are fully specified before implementation begins. All planning and implementation artifacts live under `_bmad-output/`.
