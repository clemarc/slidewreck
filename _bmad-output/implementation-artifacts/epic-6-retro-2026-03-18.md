## Successes

1. **Spike-first scoped a potential 2-epic effort down to 4 lightweight stories** — The spike correctly identified that observability is opt-in activation, not instrumentation from scratch. Epic 7 (OTEL export) was absorbed as a single config story.

2. **Clean, readable implementation** — Easy to understand for future maintainers. Constructor config, not custom span plumbing. When the underlying library is well-designed, implementation becomes trivially clear.

3. **Disciplined restraint on NFR-12** — Storage ops aren't auto-traced and can't be without upstream `@mastra/pg` changes. Team documented the gap and wrote tests proving what *is* traced, rather than hacking a workaround.

4. **Cleanest code review across all epics** — Only 1 finding (port conflict in 6.3). Config-heavy stories have less review surface area.

5. **10 new tests, zero regressions** — Observability config tests in `index.test.ts`, trace verification tests in `observability.test.ts`.

---

## Challenges

1. **Three post-story OTEL integration fixes** — TracerProvider registration (all-zero span/trace IDs), OTLP URL suffix (`/v1/traces` not auto-appended), LGTM log config. All discovered during manual Grafana verification, not caught by unit tests.

2. **NFR-12 storage tracing gap** — `@mastra/core` has 16 span types but none for storage I/O. Manual instrumentation would require upstream `@mastra/pg` changes. Documented as limitation.

---

## Key Insights

1. **Unknown-unknowns in third-party integration are expected cost, not process failures** — The OTEL fixes (TracerProvider, URL suffix) were not anticipatable during the spike. They only surface when wiring the full stack (Mastra → OtelBridge → Tempo → Grafana). The team diagnosed and fixed them quickly.

2. **When the underlying library is well-designed, implementation is easy to understand** — Mastra's observability surface (opt-in, auto-tracing, clean exporter API) made the epic straightforward. Good upstream design compounds into simple downstream code.

3. **Spike investment continues to compound** — This was the lightest non-trivial epic. The spike mapped the full surface, stories became pure execution.
